Privacy Policy

Extension Name: Turo Calendar Sync

This Chrome extension allows users to sync their Turo reservation data to Google Calendar for easy scheduling and mobile access.

What data is collected?

  • Reservation and vehicle information from the Turo website while the user is logged in.
  • Basic Google Calendar access (via OAuth) to create and manage events in the user's own account.

How is data used?

Data is used solely to generate calendar events for personal use. No data is stored or shared with third parties.

Third-party access

  • turo.com: Read-only access to calendar and reservation data while on trip pages.
  • Google Calendar API: Used to create or modify events in the userâ€™s calendar via secure OAuth2 authentication.

Data storage

No personal data is stored on any server. All processing happens locally in the browser and through secure Google API calls.

Permissions

  • identity: To authenticate the user with their Google account.
  • scripting: To extract calendar data from the active Turo tab.
  • activeTab: To identify the Turo page the user is interacting with.
  • host permissions: For accessing turo.com and googleapis.com for required API functionality.
